# food-fest-ticket-landing

## Project Overview

This project is a Vue.js ticket landing page created for the Cape Town Food Fest 2026. The aim of the project is to display different ticket tiers in a visually appealing and organised way while also including simple interactivity for the user.

This page showcases three options - Bronze, Silver, and Gold - and each ticket includes its price, a short tagline , and a list of benefits. The featured ticket is styled differently to stand out, and users can also save or favourite a ticket to create a more interative event landing page experience.

## Features

- Built with Vue.js
- Three ticket tiers displayed dynamically from local data
- Featured ticket highlighted with different styling 
- Save / favourite button interaction 
- Responsive layout for desktop and mobile screens 
- Clean event style design using resable components 

## Run

- npm run dev
